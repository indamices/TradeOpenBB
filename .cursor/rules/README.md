   ↓

   ↓

## 📊 问题分级

### 🔴 P0 - 严重（立即修复）
- 导致应用崩溃
- 安全漏洞
- 数据丢失风险
- 生产环境故障

### 🟡 P1 - 中等（尽快修复）
- 影响用户体验
- 影响代码维护性
- 性能问题
- 可能导致问题的边缘情况

### 🟢 P2 - 轻微（有时间修复）
- 代码质量问题
- 不影响功能
- 不影响用户体验
- 改进空间

## 🔍 常用扫描命令

### 快速扫描（单条命令）
```bash
# 硬编码路径
grep -r "c:\\\\Users" . --include="*.py"

# 调试代码
grep -r "console\\.log" . --include="*.tsx"

# 类型安全
grep -r "as any" . --include="*.tsx"

# 空值检查
grep -r "\\.join\(" . --include="*.tsx"
```

### 批量扫描（使用脚本）
```bash
./scripts/quick-scan.sh
```
